# Chuck norris Jokes

Creates random Chuck Norris jokes 

## Installation

```sh
composer require dndeus/chuck-norris-jokes
```

## Usage

```php
use Dndeus\ChuckNorrisJokes\JokeFactory;

$jokes = new JokeFactory();

$joke = $jokes->getRandomJoke();
```
